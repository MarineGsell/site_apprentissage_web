Cours suivi

Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web est un ensemble de pages reliées ensemble par des liens hypertexte et présentes sur Internet.

Internet

Internet est un réseau de communication mondial qui permet une communication entre les ordinateurs.

Cloud

Le cloud est un système de sauvegarde dématérialisé qui adapte sa capacité de stockage.

W3C

Le W3C est un organisme qui a pour but de définir les futurs projets du web.

Partie 2 - Distinguez les langages du Web

Langages client

Ce sont les langages utilisés pour batir l'environnement front-end d'un site web ou d'une application. Ils décrivent comment le site ou l'application doivent apparaitre.

Navigateurs

Le navigateur est une structure du web permettant la recherche de pages html.

Langages serveur

Ce sont les langages utilisés pour batir l'environnement back-end du site web ou de l'application. Ils décrivent comment le site ou l'application doivent se comporter.

Frameworks

Les frameworks sont des outils facilitant le développement front-end et/ou back-end.

Bases de données

Les bases de données stockent l'ensemble des données relatives à un site web ou une application. Elles communiquent avec la partie back-end du site via un langage qui leur est propre.

Site responsive

Un site responsive est un site capable de s'adapter à différentes taille d'écran pour pouvoir être consulté aussi bien sur un ordinateur que sur un appareil mobile.

Application mobile

U£ne application mobile est conçue pour être consultée exclusivement depuis un appareil mobile.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur sans écran qui permet le fonctionnement back-end d'un site web ou d'une application, ainsi que son accessibilité sur Internet.

Adresse IP

Une adresse IP se compose de chiffre et correspond à un numéro d'identité pour chaque site web ou application.

Nom d’hôte et DNS

Le nom d'hôte est le nom d'un site web ou d'une application. Il correspond à une adresse IP et est plus facile à retenir pour un utilisateur. Le DNS est le répertoire qui recense l'ensemble des correspondances entre adresse IP et nom d'hôte.

Protocoles

Les protocoles sont le langage universel utilisé par les machines pour communiquer entre elles.

Partie 4 - Identifiez les métiers du web

Développeurs

Les développeurs construisent les sites web et/ou les applications. Il existe plusieurs spécialisations dans le métier de développeur.

Product Managers et chefs de projets

Les products manager sont responsables de l'évolution du projet dans le temps, de la roadmap nécessaire à sa construction et de son adaptation aux besoins ds utilisateurs. Le chef de projet est reponsable de la partie technique et de l'exécution du projet définit.

Designers

Les designers assurent la construction de l'apparence du produit en veillant à son ergonomie et à sa simplicité d'utilisation.

Testeurs

Les testeurs s'assurent du bon fonctionnement du produit en effectuant des tests manuels et en mettant en place des tests automatiques.

Techniciens informatique

Les techniciens informatiques assurent la maintenant et le bon fonctionnement des outils de travail des autres métiers du web.

Boîte à outils du développeur

  • IDE comme VS Code : Un IDE est un outil décriture de code qui permet de regrouper de mannière structurée tous les fichiers nécessaire à la construction d'un site web ou d'une application.
  • Github : Github est un outilde communication et de collaboration entre développeurs.
  • Librairies / bibliothèques : Ce sont des outils renseignant le codes correspondant à des éléments visuels ou de contruction. Ils peuvent être télécharghés ou consulté en ligne, et permettent un travailplus efficace du développeur.
  • Frameworks : Les frameworks sont des outils facilitant le développement front-end et/ou back-end. Par exemple on a Symfony pour PHP, Java EE pour Java et Django pour Python.
